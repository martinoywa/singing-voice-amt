# singing-voice-amt
This research investigates whether converting human singing voice into instrument-like sounds through timbre transfer can improve the performance of automatic music transcription (AMT) systems.

# datasets
## 1. Choral Singing Dataset:
- https://zenodo.org/records/2649950
- .wav files per voice.
- .midi files per voice (Ground Truth)

## 2. VocalSet: A Singing Voice Dataset
- https://zenodo.org/records/7061507 (.csv)
- https://zenodo.org/records/1193957 (.wav)
- These extended CSVs essentially describe MIDI notes already. Can be used to generate .midi files.
